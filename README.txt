== Description ==
This is plugin will create responsive Modal Images with pure CSS and JavaScript.

== Installation ==
1. Upload `popup_image.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php do_shortcode('[Popup src="http://data.whicdn.com/images/1895077/large.jpg" alt="dog" width="400" height="300"]') ?>` in your templates.

= Does plugin shortcode working with normal post editor? =

Yes, It's working you can use the following shortcode for example [Popup src="http://data.whicdn.com/images/1895077/large.jpg" alt="dog" width="400" height="300"].

= What shortcode parameters? =

Shortcode parameters are src, alt, width, and height
